View on GitHub Chart to Data

Convert bar chart images to numeric data using a computer vision algorithm. When examining a bar chart from a report or a scientific paper without the original data source available, the options are limited to further utilize that information in your research or decision making process. There are tools that help you select the bars manually and convert them to numeric values, but these approaches suffer from scalability issues and are of questionable accuracy. This approach uses a YOLO image detection algorithm trained on an extensive collection of data containing millions of bar charts. Try Interactive Demo

View on GitHub Algorithmic Trading

Automating stock market transactions and portfolio management are of utmost interest to quantitative trading firms and hedge funds around the world. However, for small investors at least as much value is generated by these automatic decision rules simply be excluding personal judgment and bias. This trading bot is trained on a large set of financial variables using a genetic algorithm. Since automated trading cannot be framed as a supervised learning problem, the underlying NEAT model searches for a close-to-optimal rule by generating multiple generations of agents that become better at managing their own portfolios over time. The results are promising as the best agents outperform the performance of the S&P500 index.

View on GitHub Chess Engine

Chess bots are powerful examples of how well reinforcement learning is suited for situations governed by a set of rules. Chess bots are extremely advanced these days and admittedly this version is far from being state-of-the-art, however it can still make an enjoyable opponent. A simple command line interface makes it possible to interact and play against this algorithm. The model itself is based on a feed forward neural network architecture that is trained by playing against itself only knowing the rules of chess. Watch video

View on GitHub This Website

This website itself utilizes technologies that are considered industry standard in both web development and data science. The frontend is written in Bootstrap CSS and HTML while the backend logic is written in Flask. The application is exposed to the internet through Docker and Kubernetes using the Google Kubernetes Engine. While the simplicity of this application does not require complex logic in Flask or JavaScript, the Bootstrap CSS ensures that the content is responsive and visually appealing on most screen sizes. At deployment, the Dockerized application and the Kubernetes container orchestration on top of Google Cloud infrastructure make sure that the site is always up-and-running.
